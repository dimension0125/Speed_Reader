# ⚡ 스피드 독해 트레이너 (Speed Reading Trainer)

![Platform: Web](https://img.shields.io/badge/Platform-Web-blue.svg)
![Language: HTML/JS](https://img.shields.io/badge/Language-HTML%2FJS-orange.svg)
![Optimization: Responsive](https://img.shields.io/badge/Optimization-Responsive-green.svg)

> **"읽는 속도가 사고의 속도를 결정합니다."**
> 안구의 퇴행을 막고 시폭을 확장하여 국어 지문 독해 속도를 비약적으로 높여주는 미니멀 웹 도구입니다.

---

## ✨ 핵심 기능 (Key Features)

* **기기 맞춤형 반응형 디자인:** PC에서는 대화면, 모바일에서는 한눈에 들어오는 최적화된 글자 크기로 자동으로 전환합니다.
* **안구 퇴행 강제 교정:** 띄어쓰기 단위로 글자가 순차 노출되어, 이미 읽은 곳으로 눈이 되돌아가는 나쁜 습관을 고쳐줍니다.
* **정밀 속도 제어:** `0.18초`와 같이 소수점 단위로 노출 간격을 조절하여 본인의 한계 속도에 맞춰 정밀하게 연습할 수 있습니다.

---

## 🚀 사용 방법 (Quick Start)

### [🌐 해당 링크로 들어갑니다](https://dimension0125.github.io/Speed_Reader/)

1.  **지문 입력:** 연습할 국어 지문이나 칼럼을 붙여넣습니다.
2.  **속도 설정:** 원하는 노출 간격을 입력합니다. (추천: `0.16s` ~ `0.24s`)
3.  **훈련 시작:** 나타나는 글자를 리듬에 맞춰 따라가며 읽습니다.

---

## 📈 권장 훈련 단계 (Training Guide)

| 단계 | 권장 속도 | 훈련 포인트 |
| :--- | :--- | :--- |
| **1단계: 적응** | `0.25s` | 흐름을 놓치지 않고 끝까지 완독하기 |
| **2단계: 교정** | `0.18s` | 눈이 뒤로 돌아가려는 본능 억제하기 |
| **3단계: 돌파** | `0.14s` | 어절을 '이미지' 덩어리로 인식하기 |
